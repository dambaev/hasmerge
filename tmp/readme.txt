
Пример использования в test.avs

Общий смысл: добавляются 2 функции: ApplyClip и MergeClips, 
которым необходимо 3  параметра:
    base - базовый клип, к которому будут применяться все изменения
    src - клип, от которого будут браться фреймы
    scriptfile - сценарий, в котором описываются изменения, которые 
        необходимо произвести с базовым клипом

Операции, поддерживаемые сценарием
В сценарии допускается использование пустых строк, строк, 
первым символом которых является символ # - они считаются комментариями:

# произвольный текст до конца строки


Все остальные строки являются командами:

1. для ApplyClip:

* замена
строка:
r 10 109 100 199
эквивалентна сценарию:

base=AviSource("baseSource.avi")
src=AviSource("srcSource.avi")
trim(base,0,9)+trim(src,100,199)+trim(base,110,FrameCount(base))

т.е. кадры [100;199] (включительно) из src вставляются вместо кадров
[10;109] (включительно) в base. При этом, количество модет не совпадать,
т.е. возможно выкинуть, скажем, 200 кадров, а вместо них вставить 100:
r 10 209 100 199

* вставка
i 1 100 199
эквивалентен сценарию:
base=AviSource("baseSource.avi")
src=AviSource("srcSource.avi")
DeleteFrame(trim(base,0,1),1)+trim(src,100,199)+trim(base,1,FrameCount(base))

т.е. вставить между кадром 0 и 1 в base кадры [100;199] (включительно) 
из src

* удаление
d 10 11
эквивалентен сценарию:
base=AviSource("baseSource.avi")
src=AviSource("srcSource.avi")
trim(base,0,9)+trim(base,12,FrameCount(base))
т.е. выкинуть с 10 по 11 кадры (включительно) из base

* добавление
a 100 199
эквивалентен сценарию:
base=AviSource("baseSource.avi")
src=AviSource("srcSource.avi")
base+trim(src,100,199)
т.е. добавляет в конец base кадры [100;199] (включительно) из src


2. для MergeClips

MergeClips оперирует понятиями левого (первый параметр) и правого клипа
(второй параметр):

MergeClips(left,right,"script.txt")

2 команды: l и r, что означает кусок из левого клипа и кусок из правого 
соответственно

* кусок из левого

l 100 200
эквивалент:
left=AviSource("baseSource.avi")
right=AviSource("srcSource.avi")
trim(left,100,200)
т.е. вставить диапазон фреймов [100;200] из левого клипа

l 100
эквивалент:
left=AviSource("baseSource.avi")
right=AviSource("srcSource.avi")
trim(left,100,-1)
т.е. вставить фрейм 100 из левого клипа

* кусок из правого клипа

r 100 200
эквивалент:
left=AviSource("baseSource.avi")
right=AviSource("srcSource.avi")
trim(right,100,200)
т.е. вставить диапазон фреймов [100;200] из правого клипа

r 100
эквивалент:
left=AviSource("baseSource.avi")
right=AviSource("srcSource.avi")
trim(left,100,-1)
т.е. вставить фрейм 100 из правого клипа


При этом, в данном описании эквивалентные сценарии приведены 
обособленно, для случаев, когда в сценарии команд указана 
всего 1 операция. 

В случае, когда команд несколько, то для каждой следующей команды 
клип base будет результатом выполнения текущей команды, так, например:
i 1 100 199
d 1 99
сначала добавит кадры между кадрами 0 и 1, а затем удалит их же, т.е.
будет равносилен отсутствию изменений.
